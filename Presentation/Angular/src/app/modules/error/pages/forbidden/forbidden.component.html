403 - ACCESO DENEGADO
Oops, No tienes privilegios para acceder a esta página
Un servidor web puede devolver un código de estado HTTP 403 Forbidden en respuesta a una solicitud de una página web o recurso por parte de un cliente para indicar que el servidor es accesible y ha entendido la solicitud, pero que se niega a seguir adelante.

Para indicar que el servidor puede ser alcanzado y entendió la solicitud, pero se niega a tomar cualquier otra acción.

Las respuestas con código de estado 403 son el resultado de que el servidor web está configurado para denegar el acceso, por alguna razón, al recurso solicitado por el cliente.
